# POG Streamlit アプリ 設計書

**作成日**: 2026-04-17  
**対象ユーザー**: ぷに（ぽぐ！ UserId=8）  
**目的**: ぽぐ！の指名リスト作成を効率化し、称号狙い候補の自動検出とサイトへの自動登録を実現する

---

## 概要

Pythonで動作するStreamlitウェブアプリ。`streamlit run app.py` で起動し、ブラウザで操作する。

主要機能：
1. 全7,864頭の馬リストをフィルタ・検索
2. 過去指名馬との血統つながり（称号狙い候補）を自動ハイライト
3. 牡50頭・牝50頭の指名リストをドラッグで優先順位付け
4. ぽぐ！サイトへワンクリック自動送信

---

## 画面構成

### ① 馬リスト画面（メイン）

**表示項目**
- 馬名、性別、父名、母名、母父名、産地、調教師、取引価格
- 称号候補フラグ（🏆）と称号候補の説明（例：「チェルヴィニアの弟」）

**フィルタ**
- 性別（牡/牝）
- 父名（テキスト検索）
- 産地（北海道/本州など）
- 取引価格（スライダー）
- 血統系統（ロイヤルチャージャー系など）
- 称号候補のみ表示（チェックボックス）

**操作**
- 各行の「リストに追加 ＋」ボタンでマイリストへ追加

---

### ② 称号狙い専用画面

ぷにさんが過去に指名した馬の**父・母・兄弟姉妹**を持つ馬を自動抽出。

**表示内容（例）**
```
🏆 チェスティーノ（エフフォーリア × チェッキーノ）
  → 2015年にぷにが「チェッキーノ」（母）を3番指名
  → 姉「チェルヴィニア」がオークス優勝！
```

**データソース**
- ぽぐ！の advice.php（ログイン済みスクレイピング）
- 過去20年の指名履歴JSON（ローカルキャッシュ）

---

### ③ マイリスト画面

牡リスト・牝リストを並べて表示。ボタンで順位を上下入れ替え。

**表示**
- 牡リスト（最大50頭）/ 牝リスト（最大50頭）
- 上位5頭を「指名確定予想」として緑背景で強調
- 各馬の称号候補フラグも表示

**操作**
- 「↑↓」ボタンで順位変更
- 「削除」ボタンで除外
- 「ぽぐ！に送信 🚀」ボタンでサイトに自動登録

---

### ④ 分析画面

- 過去20年の父別成功率グラフ（棒グラフ）
- 年別獲得ポイント推移
- 当たり馬の特徴サマリー（父・産地・厩舎の傾向）

## 自動送信の仕組み

1. ログイン：`EntryLoginId=8 & EntryLoginPassword=sika` をPOST
2. セッションクッキーをpickleに保存
3. リスト送信：`draftEntry.php` に馬IDの順序付きリストをPOST
4. 成功/失敗のレスポンスをUI上に表示
